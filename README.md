Creating and Managing Chats

- Create a Chat
  ```bash
  skunk chat create <chat-name>
  ```
- Invite Peers
  ```bash
  skunk chat invite <peer-username>
  ```
- Join a Chat
  ```bash
  skunk chat join <chat-name>
  ```
- Leave a Chat
  ```bash
  skunk chat leave <chat-name>
  ```
